Sistema de Cadastro de Produtos em C
Descrição

Este projeto foi desenvolvido em linguagem C com o objetivo de praticar conceitos fundamentais de programação, como:

Vetores
Estruturas de repetição (while e for)
Condicionais (if)
Manipulação de strings
Entrada e saída de dados
Cálculos e lógica de programação

O programa permite cadastrar até 10 produtos, armazenando:

Nome do produto
Preço unitário
Quantidade em estoque

Ao final do cadastro, é gerado um relatório contendo:

Lista de todos os produtos cadastrados
Preço de cada produto
Quantidade em estoque
Valor total em estoque por produto
Maior valor de estoque encontrado
Quantidade total de itens em estoque
Valor total do estoque
Tecnologias Utilizadas
Linguagem C
Biblioteca stdio.h
Biblioteca string.h
Como Executar
Compile o programa:
gcc main.c -o produtos
Execute:
./produtos


Objetivo do Projeto

Este projeto foi desenvolvido como exercício de aprendizado durante os estudos de programação em C, com foco em lógica de programação e manipulação de vetores.
